What is Alpha Blending?
알파 블랜딩 : 배경 이미지 위에 투명도가 전경 이미지를 오버레이하는 프로세스
투명도 마스크는 종종 알파 마스크 or 알파 매트라고 한다.

알파 블렌딩 이면의 수학은 간단하다.
이미지의 모든 픽셀에서 알파 마스크(\aplha)를 사용하여
전경 이미지 색상과 배경 이미지 색상을 결합해야 한다.

AX + BY = 섞인 물감
X + Y = 1
-> AX + B(1-X) = 섞인 물감

// X = 0 이면, 출력 픽셀 색상은 배경생입니다.
// X = 1 이면, 출력 픽셀 색상이 전경입니다.
// 0<X<1 이면, 출력 픽셀 색상이 배경과 전경의 혼합입니다.
-> 사실적인 블렌딩을 위해 알파 마스크의 경계애는 일반적으로 0과 1사이의 픽셀이 있다.
