1. OpenCV(Open Source computer Vision)
: 영상 처리에 사용할 수 있느 오픈 소스 라이브러리
: 컴퓨터가 사람의 눈처럼 인식할 수 있게 처리해주는 역할
: 왜곡정보를 올라르게 해석하기 위한 도구를 제공해주는 것을 목표
// 카메라로 찍어서 할 수 잇는 일을 Open로 처리 가능

: BSD(Berkely Software Distriburtion)
  컴퓨터 비전과 인공지능의 발달 시키고자하는 바램으로 출시 함.

2. 컴퓨터 비전 
: 카메라로 얻은 일련의 영상들
: 사람에게 쉬운일은 컴퓨터에게 어려울 수 있다.
: 컴퓨터는 숫자의 형렬로 사진외의 형태로 인식

3. OpenCV 기본 구조
CV + MLL + HighGUl -> CXCORE
CV : 이미지 처리와 비전 알고리즘
MLL : 통계적 분류 및 클러스터링 도구
HightGUI : GUI, 이미지 and Video I/O
