2차원 배열

바둑판에 놓여있는 돌의 정보를 저장하려면?
1. 돌 한개의 정보를 어떤 형색의 변수에 저장할 것인지를 결정
    0 : 돌이 놓여있지 않은 경우
    1 : 검은 돌이 놓여 있는 경우
    2 : 흰 돌이 놓여있는 경우
2. 바둑판 전체를 저장할 수 있는 메모리 크기를 결정

2차원 배열의 필요성
-> 2차원 공간에 잇는 대상은 자신의 위치를 좌표로 나타내는 형태로 사용
-> 바둑판에 높은 바둑돌의 위치 정보도 2차원 형식의 데이터
-> 2차원 배열 선언하기 - 행(Y축)단우이로 묶음
-> 배열 문법을 사용하여 논리적으로 재구성한 2차우너 데이터 형식

// 1차원 배열과 2차원 배열의 차이
