1. 계정 생성 -> sudo adduser ubuntu
2. 계정 검색 -> cat /etc/passwd | grep ubuntu
/etc/passwd : 사용자 인증에 필요한 계정 정보
/etc/shadow : 사용자 인증에 필요한 암호 정보(암호화)
3. 계정 수정 -> 계정명을 받아 옵션에 적용된 값의 지정
usermod[옵션][계정명] : passwd[계정명]
4. 계정 삭제 -> 생성했던 사용자 계정을 삭제
userdel[옵션][계정명]
