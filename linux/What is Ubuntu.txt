02. 우분투(Ubuntu)에 대해서

우분투(Ubuntu)란?
1. 컴퓨터 운영 체제로서 고유한 데스크탑 환경인 유니티를 사용하는 리눅스 배포판
2. 개인용 데스크톱과 노드북에 가장 인기 있는 배포판
3. 시스템 관리 작업에서 #sudo 도구를 사용한다.
-> 암호를 통해 관리 권한을 얻어 작업을 진행할 수 있도록 인증한다.
-> 여러 사용자 관리를 위해 root 암호를 공유함으로써 생길 보안문재를 예방

우분투 서버 버전?
APT 저장소를 사용하는 서버 판 우분투를 가리킨다.
서버판은 설치 과정에서 그래픽 인터페이스 대신 커맨드 라인 인터페이스를 자원한다

Install - 서버 버전
1. Download
2. Server
3. Ubuntu Server 20.04 LTS Download
4. 한국어 -> 우분투 서버 설치
5. Korean-Korean(101/104 Key compatible)

ppa-purge 
: PPA를 통해 설치한 모든 프로그램 삭제 저장소명을 이용한 삭제가 가능합니다.
: ppa의 프로그램,패키지,저장소 삭제의 번거로움을 해결하는 패키지를 의미함.
PPA(Personal Package Archive)란?
-> 우분투 소프트웨어 센터에 있는 저장소는 안전하고 안정적이다.
-> 최신버전의 경우 다음 버전까지 기다려야하는 상황이 생긴다.
-> 우분투에 포함되어 있지 않지만, 우분투보다 빠르게 업데이트가 가능함
// 우분투 소프트웨어 센터에 등록되지 않는 프로그램 설치할 때 사용한다.
