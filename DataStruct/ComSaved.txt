컴퓨터의 데이터 저장 방법
[자료구조 이해는 데이터 관리의 효율성을 높인다.]

컴퓨터의 데이터 저장소
1. 스토리지 -> HDD, SDD
- 데이터를 영구적으로 보관 및 많은 양의 데이터 저장[느림]
2. 메모리 -> RAM
- 데이터를 일시적으로 보관 및 빠른 속도로 데이터 처리
// 메모리는 스토리지보다 훨씬 빠른 속도록 데이터 처리함

데이터 저장소의 특성
RAM이 어떠한 방식으로 데이터를 저장하는 방식을 알게되면,
서비스 및 프로그램의 작동 및 처리를 효율적으로 관리 가능

RAM(Random Access Memory)는 임의접근 메모리
-> 순차접근 [처음부터 끝까지 거쳐야 데이터를 아는 것]
-> 임의접근 [해당 지점 주소만으로 데이터 접근하는 것]
